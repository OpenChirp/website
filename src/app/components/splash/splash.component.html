About

What is OpenChirp?

OpenChirp is a management layer on top of LoRaWAN that provides
data context, storage, visualization, and access control over the web.
OpenChirp is currently powering a LoRaWAN Network at CMU that
also spans most of the City of Pittsburgh. You can expand the OpenChirp
network by using our software to connect your own LoRa base station.

Technology

Features

OpenChirp is built on a standard MEAN (MongoDB, ExpressJS, AngularJS, Node.js) stack.
The system uses MQTT for its publish-subscribe layer and InfluxDB for time series data storage,
but it flexible enough that application specific components can be added as services.
Checkout our Github page for our software repository and API documentation.
